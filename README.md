# AI/ML Enabled Video Analysis and Interpretation System (NSG)

This project is an automated intelligent surveillance software designed for the National Security Guard (NSG) under the Ministry of Home Affairs (MHA). 

It acts as a digital force multiplier at the Incident Command Post (ICP) by automatically analyzing and interpreting visual feeds from multiple devices—such as body cams, tactical drones, and scout robots—without human intervention.

## Core Capabilities

1. **Real-time Video Feed Processing**: Receives and processes multiple simultaneous video feeds from legacy and modern cameras.
2. **AI-Powered Object Detection**: Uses neural networks to detect security-relevant objects, focusing on weapons, bags, and items of interest.
3. **Facial Recognition Watchlist**: Detects and recognizes faces, comparing them against a registered watchlist of high-priority individuals.
4. **Behavior and Threat Alerts**: Monitors activity patterns (such as loitering or restricted zone entry) and automatically raises visual alerts.
5. **Interactive Analytical Dashboard**: Displays live video feeds, real-time alert logs, threat hotspots, and a manager to enroll faces into the security watchlist.
